# 📐 Conversor de Unidades TecNM

Aplicación de escritorio desarrollada en **C# (.NET)** para la conversión de unidades de medida, desarrollada en el contexto académico del **Tecnológico Nacional de México (TecNM)**.

---

## 📋 Descripción

**Conversor de Unidades de monedas TecNM** es una herramienta de escritorio que permite convertir entre monedas de forma rápida y sencilla. Ideal para uso académico e ingenieril dentro del entorno del TecNM.
